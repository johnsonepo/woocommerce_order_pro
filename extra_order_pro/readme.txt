## Description

Take control of your WooCommerce orders with the feature-rich WooCommerce Order Extras plugin. This extension introduces additional fields such as delivery agents, payment methods, and payment status, ensuring data integrity. Lock orders with confirmed payments, maintain account consistency, and prevent accidental deletions. Elevate your order management for greater control and reliability.

## Installation

1. Upload the 'woocommerce-order-extras' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Ensure WooCommerce is installed and activated.

## Features

- Adds custom order status functionality.
- Integrates delivery agent information.
- Adds support for custom payment methods.
- Enhances WooCommerce order management.
- Enables the option to require a payment method for each order.

## Frequently Asked Questions

**Q: Does this plugin work without WooCommerce?**  
A: No, this plugin requires WooCommerce to be installed and activated.

**Q: How to access custom order status, delivery agent, and payment settings?**  
A: Navigate to the WooCommerce menu in the admin panel. Find submenu options for 'Custom Order Status,' 'Delivery Agent,' and 'Payment Extra.'
